# Strony-Internetowe

Projekt strony internetowej dla Futomaki - agencji produkującej materiały wideo dla marketingu.

## Opis

Strona zawiera sekcje:
- Hero z opisem usług
- O nas
- Dlaczego Futomaki (korzyści)
- Jak wygląda współpraca (placeholder)
- Portfolio (placeholders dla projektów)
- Kontakt z formularzem

## Technologie

- HTML5
- CSS3 (responsywny design)

## Jak uruchomić

Otwórz `index.html` w przeglądarce internetowej.

## Pliki

- `index.html` - główna strona
- `styles.css` - arkusz stylów
